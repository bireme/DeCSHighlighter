DeCS <em>Highlighter</em>:

Description:

A web service to highlight (insert xml tags) all DeCS descriptors and synonynms that occur in an input text.

Example:

Given an input text:

"Pharmacotherapy has a pivotal role in the management of endometriosis with long-term treatments balancing clinical efficacy (control of pain symptoms and prevention of recurrence of the disease after surgery) with an acceptable safety profile. Treatment choice is based on several factors including age and patient preference, reproductive plans, intensity of pain, severity of disease and incidence of adverse effects.Areas covered: The aim of this review is to provide the reader with a complete overview of drugs that are currently available or are under investigation for the treatment of endometriosis highlighting on-going clinical trials."

The output text could be like that:

"Pharmacotherapy has a pivotal role in the management of endometriosis with long-term treatments balancing clinical efficacy (control of pain symptoms and prevention of recurrence of the disease after surgery) with an acceptable safety profile. Treatment choice is based on several factors including age and patient preference, reproductive plans, intensity of pain, severity of disease and incidence of adverse effects.Areas covered: The aim of this review is to provide the reader with a complete overview of drugs that are currently available or are under investigation for the treatment of endometriosis highlighting on-going clinical trials."

The web service:

Path: http://<host>:<port>/serv

Methods: GET or POST

Consume: application/x-www-form-urlencoded

Produce: application/json

Parameter:

  document= the input text to be highlighted

Optional parameters:

  prefix= the text prefix to be used before any found descriptor/synonym. Default value is '<em>'

  suffix= the text suffix to be used after any found descriptor/synonym. Default value is '</em>'

  showText= outputs the highligted text

  showPositions= outputs the begin and end positions of each descriptor/synonym and the respective DeCS identifier

  showDescriptors= outputs the list of found descriptor/synonym(s)

If showText, showPositions and showDescriptors parameters are all absent, then the text, positions and descriptors are displayed.


Example input:

  http://<host>:<port>/serv?prefix=<font color='blue'>&suffix=</font>&document=the malaria desease kills.

Example output:

          {
            "text" : "the <font color='blue'>malaria</font> desease kills",
            "positions" : [
              {
                "begin" : 4,
                "end" : 10,
                "id" : "008462",
                "descriptor" : "malaria"
              }
            ],
            "descriptors" : [
              "malaria"
            ]
          }

Where:

  text - outputs the highlighted input text

  positions - indicates the begin and the end positions for each descriptor/synonym in the input text and its DeCs indentifier

  descriptors - shows each descriptor/synonym found.

Example application:

Description: A web application that displays the highlighted input document.

Path: http://<host>:<port>/app
